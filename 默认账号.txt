注意事项：
========================================

1. 首次使用请先初始化数据库
   运行：server/database/init-database.bat

2. 修改数据库配置
   编辑：server/.env
   修改 DB_PASSWORD 为你的 MySQL 密码

3. 启动项目
   方式1：运行 启动项目.bat
   方式2：手动启动
      - 后端：cd server && npm run dev
      - 前端：npm run dev

4. 访问地址
   前端：http://localhost:5173
   后端：http://localhost:3000

5. 生产环境部署
   - 修改 JWT_SECRET 为复杂密钥
   - 修改默认管理员密码
   - 配置 HTTPS
   - 配置防火墙规则
